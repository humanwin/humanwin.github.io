# 本节主题：讲的是作为交易者要怎么定位自己
# 波动与趋势的区别
## 波动现象
1. 很多实盘大赛里面，一味做所谓追涨杀跌的涨停板战法的，很容易陷入这样的情况：
    - 有一段时间，很快做到翻倍。
    - 然后再过一段时间，发现又跌下来。
    - 看着很牛逼，最后一算账，发现好像没赚到多少钱。
## 波动现象的误区
1. 认为自己只打出波动，控制不了回撤的原因是技术还不到位。
2. 更加努力的钻研技术，然后发现始终解决不了这个问题。
3. 这个问题的根本原因不在术，而是在道上。
## 个人账户不同年收益率的对应表现
1. 个人账户收益率达到多少，你会挺满意，家人朋友问你，你会愿意说？
2. 根据盘哥经验，情况如下：
    - 亏钱：聊起来会说不好做，但心里暗暗发誓，要赚10倍让人闭嘴。
    - 超过20%：聊起来会愿意说，但是会表达没做好。
    - 超过30%：会特别愿意说。
    - 超过50%：甚至会主动说。

# 对股市赚钱的概念
## 对赚钱有概念
1. 很努力的做了1年之后，收益达到30%，虽然不满意但是最终接受。
2. 发现原先1年10倍的预期是不合理的，但内心相信只要时间够长，不管是1年还是3、5年，总会到10倍。
3. 相信只要赚钱就是好的，开始进入正循环。 
## 对赚钱没概念
1. 预期股市1年赚10倍的，都属于对赚钱没概念。
## 还在亏钱但仍想1年赚10倍的心理
1. 处于极度自卑的状态。
2. 有一颗不平凡的心，特别希望自己是个不平凡的人，但现在没有能力展示自己的不平凡。
3. 想通过敢于去做别人不敢做的事，来向身边的人展示自己很牛逼。
4. 他的一切骄傲，不在于做成了什么事情，而在于敢去做别人不敢做的事情。
## 预期太高的弊病
1. 因为预期太高，导致自己都不相信能做成这件事，所以不会很认真的去做这件事。
## 正确的股市预期
1. 人对自己的骄傲要放在你做成了什么事上，而不是你要去做什么事上。
2. 不能有龙头战法的优越感，赚钱才是真，没赚钱都是浮云。
    - 很多做龙头股的人最后会陷入死循环，会合理化自己的亏损，认为自己成功能赚10倍，不成功亏个20%也合理。
3. 一个合理的预期目标，是经得起一步一步的拆解的。
4. 一个预期目标的实现，是需要一步一步拆解实现的。
## 一年10倍的拆解假象
1. 一年的每个月，都实现单月22%的盈利，一年就可以10倍收益。
2. 现实上，如果一个基金经理能够实现10年以上的年复利20%（注意是年哦），那他已经是世界顶级投资选手了。
3. 很多人混淆了灵活性和持续性的概念，误认为：“这些基金经理是因为资金规模大，所以持续收益率低。小资金不会，小资金可以实现持续的收益”。
## 大资金与小资金的差别
1. 大资金：持续性好，灵活性差。
2. 小资金：灵活性号，持续性差。
### 持续性：小资金和大资金的理解
1. 在大的趋势行情里，小资金反而会有劣势，甚至跑不过公募基金。
2. 因为在趋势向上的过程中，有进入正常调整的时候，小资金灵活，回调就卖掉了。
3. 但回调可能是超预期回调，只1天就涨回去了，但小资金此时犹豫了，认为可能是假反包或者是其他的。
4. 在这种趋势向上的回调过程中，小资金不断的去操作，最后发现交易磨擦成本变得非常大。
5. 而大资金因为灵活性不足不好进出，反而拿住这种趋势性行情。
6. 这种灵活性和持续性的限制，不是能够通过技术去解决的，这是一个天然的问题。

# 刚刚入门的选手怎么规划自己的交易收益预期是健康的？
1. 单月收益为正，及格。
2. 单度收益超过5%，成功。
3. 单度收益超过10%，极度成功。
4. 单月收益超过20%，则获得一次谨慎博弈高风险交易的资格。
5. 单月出现高点回撤超过5%，则提高警惕，转入保守。
6. 单月出现高点回撤超过5%，则全面转向保守。
7. 单月出现亏损5%，则全面转向保守。
8. 单月出现亏损10%，，则只允许做极度确定性交易，且若下一笔依旧不盈利，则单月不在允许交易。
## 复利的收益
1. 如果持续单月收入5%。
    - 一年收益大概80%。
    - 三年收益大概5倍。
    - 五年收益大概18倍。
2. 如果持续单月收入10%。
    - 一年收益大概2倍。
    - 三年收益大概30倍。
    - 五年收益大概300倍。
## 很多人的误区
1. 觉得自己资金量特别小，一年2倍对你来说改变不了什么。
2. 五年300倍，对你来说时间太长，你等不了。你还是希望一年10倍，20倍。
    - 想一年10倍就得全仓不停的干龙头，但因为有很多随机性在，就算你技术再高，最后大概率也是颗粒无收。
## 正确的自恃和坚持
1. 对于当下的期许期望，把自己当成一个普通人。在当下，做普通人都能做到的成就，然后坚持做下去。
2. 普通人和牛逼的人真正的差别不在于某个阶段的爆发差别，而在于他们是否能够坚持。
## 赚快钱的方式
1. 谁都想要暴利，没有人不想要赚快钱，但赚快钱的真正奥义是：正确路上行大运。
2. 健康的一年10倍与一年1倍，背后所需的技术，理念，认知是一样的。唯一的差别在于随机的运气和行情是否支持。
3. 如果在熊市还不能实现不亏钱，牛市很难赚到大钱。
4. 先把追求单月不亏钱，当成自己的目标，有可能今年意外赚10倍。

# 风险博弈的尝试资格
1. 都说牛股都是试出来的，所谓试，不就是确定性比较差的交易？
## 可能性交易
1. 确定性不高，但收益率可观，高收益，低确定性。
2. 比如成了赚20-30%，输了亏10%。
## 可能性交易的尝试资格
1. 要有规范自己的交易，就要约束自己做可能性交易的资格。
2. 要先完成单月基础的目标才可以获得资格，拿一部分利润来博可能性的交易。
3. 如果做成了单月收益率就上去了继续拿一部分收益博，如果没成就重新完成基础目标。 
4. 单月收益就是单月收益，下个月要重新开始。单月就算完成60%或70%也是单月收益，下月也不可以浪，也要重新开始。
## 风险博弈资格管控案例
1. 本月全仓收益达到2%，可以用1成仓位去试1次风险博弈（比如特别看好的龙头博弈）。
2. 本月全仓收益达到20%，可以重仓去是试1次确定性比较高的风险博弈。
    - 博对了，就有超额收益。
    - 博错了，收益回撤了5%，就不要干了，也已经差额完成了月度目标。

# 先练成这招：多看少买（最好先看3年）
1. 你一个月都在非常认真的研究股市寻找机会，但是你不买。
2. 你要想清楚，你是想很刺激很爽的输，还是很无聊的赢。
3. 怎样能实现不亏钱？最简单的方法就是不买票。不买不亏钱是及格60分。
4. 一个月只做5笔，你认为确定性很高的交易。那么盈利在10%以上就不是太难的事。
---
# 对于回撤的警惕性
1. 当你发现自己亏钱的时候，很可能是刚刚才开始亏钱。
2. 很多大回撤，都是从大赚之后小回撤，不服气，急于打回高点，然后一步一步被套进去，最后不但盈利全亏光，还开始爆亏。
3. 越是初期亏损，越喜欢做可能性交易，想证明自己技术很好。但其实技术好不好是体现在如何判断确定性交易。
4. 从高点回撤5%，就不要再做可能性交易。
5. 从高点回撤10%，就全面转向保守，最好本月不要交易。
6. 因为从高点回撤10%后，此时你肯定不服气，特别想扳回来，交易心态已经非常激进。做下去可能失去全部的收益。
## 回撤的发展
1. 回撤这种事，初期是跟运气有关。
2. 后期会扩大到状态，比如连亏两笔，状态就开始不好了。
3. 再往后就会影响信心和心态，比如连亏三笔。
### 交易变型（上头）的案例
1. 单月已经赚了20%，博一把回撤了5%。
2. 首先，你不认为亏5%有多了不得，会给交易找各种理由，因为前面挣了20%，信心还是高度膨胀的。
3. 然后，你特别不服气，想赶紧给扳回来。此时交易已经变型了。
4. 上头是没有意识到自己上头，但行为开始变了，开始不讲纪律，不讲规则。
---
# 亏损应对
1. 如果你本月很不顺，一开始就出现亏损，慢慢爬沟吧。
2. 爬沟最好的方式，就是慢慢做确定性交易。
3. 千万不要指望在高波动的可能性交易上去博一把来回水，这个风险极大。
4. 真正的，只有时间才能让你从亏损的心态里出来。
## 亏损应对案例
1. 如果本月亏了5%，本月目标就是争取不亏钱。
2. 如果本越亏了10%，接下来只允许做1笔极度确定的交易，而且这笔交易如果还是亏的，这个月不准做了，别给自己找任何理由。

# 关于单次交易&买入&卖出
## 单次交易
1. 错过、卖飞、挨了亏损，这都正常，不决定我们的收益。
2. 一个票多挣10%和少挣10%长期来看没有影响。
3. 在这笔交易里格局多挣了10%，必然会在未来，同个模式里面，因为格局亏掉10%。
4. 决定收益的是可持续化的交易模式的执行。
## 买入
1. 预判式的买入龙头。
2. 根据板块联动程度，打前排板。
3. 根据板块联动强度，低吸后排性价比套利标的（300，中军，辨识度票）。
4. 如果因为手慢，或者对板块强度不确认，或者没有性价比票，就不买，等待下次机会。
## 卖出
1. 尽早的结束一笔失败的交易。
2. 只要盈利卖出，一定没错。
3. 板块走弱，个股不强板，卖出一定没有错。
4. 炸板如果回封，但封住后，封单在撤单，很可能会炸，此时卖掉是正确的，没有问题。
## 交易心态
1. 重仓买入后，没有人会很轻松跟放松，都是走一步看一步。
2. 交易是一个非常私人的事情，没法分享。

# 一个逻辑的`持股观察周期`
## 如何判断要不要格局
1. 去总结基于个股本身的东西没有价值。
2. 只需要框架性的理解就行了。
    - 比如干了最强板块，是个后排或者涨停被砸了，是个失败票。
    - 个股虽然失败了，但只要逻辑并没有失败，明天看看板块怎么样。
    - 只要板块走强，当天可以看一看，如果脉冲回水了，看他一波有没拉板，如果没有，就尽早结束这笔失败的交易。
    - 接下来再去干下一笔。
3. 在有盈利的情况下，要一直让自己处于一个非常主动的态势，觉得逻辑上是中线，可以熬一熬。
4. 如果是亏损的情况下，别讲什么趋势、逻辑，尽早结束，有反弹就跑，别觉得卖飞什么的，你介入的时点不对。
## 卖出案例
1. 20220331，做高标票，盘龙药业。
2. 开的比较低，有性价比，就买入了。
3. 下午跳水，回落了，被套6个点，这笔交易失败了。
4. 但是近期高位龙头会有反复，可以明天(20220401)开盘再看一下。
5. 开盘如果下杀，并在水下震荡，就割了不做念想。
6. 但开盘不久就走强拉高解套了，就看这一波是否拉板，如果没有拉板开始回落，立马走人。
7. 至于之后是否又拉板反包，和本次交易没有关系。当没做过这笔交易。

## 趋势反思
1. 发现某个板块整体在转暖，但不是太有信心能走出强的趋势。
2. 很早介入，涨了但也没涨那么强。买入后有盈利但没有涨停，是否该卖？
3. 如果你想的是一个板块趋势，那持股观察周期就要变长了。
4. 可以放宽到跌破5日线或者格局到成本线2个点以上再卖。只有这样才能拿住一波趋势。

# 盘哥直播用法
1. 皆明报盘主要是为了讲课，是为了提醒板块加强，龙头股早就已经涨了，不能依赖皆明打龙头股或前排，顶多看看有没套利低吸机会。
2. 龙头股玩法
    - 每日盘前逻辑都已经梳理好，把给的股票池按逻辑放到板块里，观察联动。
    - 比如集合竞价发现开的比较不错，就要根据自己的交易模式，决定要不要去干龙头。
